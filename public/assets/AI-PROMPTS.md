# AI render prompts (hybrid asset strategy)

The site currently uses real CC stock photos in `assets/img/`. To upgrade
any slot to a bespoke, perfectly on-brand render, generate with these
prompts (Midjourney / Firefly / Imagen / DALL·E), export **16:10, dark,
cinematic**, and save over the matching filename — no code change needed.

House style to append to every prompt:
> cinematic, dark graphite tones, dramatic rim lighting, volumetric haze,
> amber + electric-cyan accents, photoreal, ultra-detailed, 16:10, no text.

| File | Prompt seed |
|------|-------------|
| `refinery.jpg` | A modern bio-bitumen refinery at dusk — distillation towers, pipelines, storage tanks, glowing amber lights |
| `biomass.jpg` | Baled agricultural crop residue (rice straw, bagasse) in a field at golden hour, feedstock for pyrolysis |
| `pyrolysis.jpg` | Interior of a high-tech pyrolysis reactor hall, glowing furnace, stainless pipework, engineers in PPE |
| `recycling.jpg` | Plastic-waste sorting and conversion plant, conveyor of shredded plastic feeding a reactor |
| `tyres.jpg` | Neatly stacked end-of-life tyres at a rubber-to-oil pyrolysis facility |
| `datacenter.jpg` | Futuristic data centre aisle with cyan-lit server racks, holographic AI dashboards overlay |
| `finance.jpg` | Bloomberg-style trading terminal wall, candlestick charts, market heatmaps, dark room |
| `sustainability.jpg` | Solar field beside a green industrial plant, carbon-neutral infrastructure, dawn light |
| `hero.jpg` | Wide establishing shot of an industrial bio-bitumen complex at blue hour, cinematic scale |
